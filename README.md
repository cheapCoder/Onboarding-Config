# 我个人新入职拿到电脑时要配置什么

- 下载搜狗输入法，setting 禁用

- 安装[brew](https://brew.sh/)

- 安装 tree 命令
  ```bash
  brew install tree
  ```
  
- 安装 zsh，oh-my-zsh 及其插件，[powerlevel10k 主题](https://www.poloxue.com/posts/2023-10-16-zsh-themes-and-plugins/#%E9%AB%98%E7%BA%A7%E7%9A%84%E4%B8%BB%E9%A2%98%E5%92%8C%E6%8F%90%E7%A4%BA%E7%AC%A6%E5%AE%9A%E5%88%B6)
